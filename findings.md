# 项目发现与研究

## 设计参考

### 精致产品感的关键要素
1. **留白** — 充足的 padding/margin，避免拥挤
2. **字体层次** — 通过字号、字重、颜色区分信息层级
3. **微交互** — 细腻 hover 态、过渡动画
4. **阴影克制** — 轻微阴影制造层次，不滥用
5. **色彩一致性** — 严格遵循配色系统

### 优秀简历网站特征
- 3 秒内传达核心信息（你是谁、做什么）
- 可扫描式布局（面试官平均停留时间短）
- 明确的 CTA（联系方式突出）
- 移动端友好（很多面试官用手机查看）

## 技术选型依据

### Vite 优势
- 极速启动和 HMR
- 开箱即用的 TypeScript 支持
- 优秀的生产构建优化
- 轻量、现代

### React 选择
- 组件化思维展示
- 便于状态管理（如主题切换）
- 展示现代前端技能
